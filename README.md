# NDA Property - Website UMKM Properti & Arsitektur

Website modern, interaktif, dan responsif untuk NDA Property - UMKM yang berfokus pada desain, konstruksi, renovasi, manajemen proyek, dan pemasaran properti.

## 🎯 Fitur Utama

- **Hero Section**: Animasi blueprint arsitektur dengan karakter arsitek
- **About Section**: Profil perusahaan dengan statistik dan misi
- **Services Section**: 6 layanan utama dengan hover animation
- **Portfolio Section**: Galeri proyek dengan filter kategori
- **Marketplace Section**: Listing properti dengan filter pencarian & sorting
- **Testimonials Section**: Review klien dalam carousel
- **Contact Section**: Form konsultasi & informasi kontak lengkap
- **Dashboard Admin**: (Coming soon) Manajemen properti untuk admin

## 🛠️ Tech Stack

- **Framework**: Vue.js 3
- **Styling**: TailwindCSS
- **Build Tool**: Vite
- **Animation**: CSS Keyframes + Vue Transitions

## 📋 Instalasi & Menjalankan

### Prerequisites
- Node.js 16+ 
- npm atau yarn

### Steps

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Development server akan berjalan di `http://localhost:5173`
